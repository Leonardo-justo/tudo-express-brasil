# Tudo Express Brasil

Landing page estática, responsiva e otimizada para apresentar a marca e encaminhar visitantes às lojas oficiais.

## Visualização

Abra `index.html` diretamente no navegador ou sirva a pasta com qualquer servidor HTTP local.

## Estrutura

- `index.html`: conteúdo e SEO
- `styles.css`: identidade visual e responsividade
- `script.js`: menu móvel, animações e ano automático
- `assets/`: imagens fornecidas pelo cliente
